# Projeto_Final_Prog2  
Este é o projeto final da disciplina Programação 2, onde é desenvolvido um sistema para administrar a frota de veículos da UENP, usando a linguagem Java e principalmente Programação Orientada a Objetos.  
Para rodar esse projeto, clique com o botão direito no nome do projeto na aba Projects no NetBeans e vá em Run Maven e depois Other Goals e escreva "clean compile javafx:run".    
O banco de dados usado para esse projeto foi o MongoDB, usando O MongoDB Compass para melhor visualização.
